## 中文说明

这个仓库 `baokaixina/simple-translation-mod` 保留的是早期 **Simple Translation 1.0** 代码和发布记录。

当前维护中的新版项目是 **SimpleTranslate 2.0 / 简单翻译 2.0**，已经迁移到：

<https://github.com/baokaixin/SimpleTranslate>

请优先前往新仓库下载、查看说明或提交反馈。

### 新版 2.0 的主要变化

- 使用 Minecraft **Component JSON** 翻译管线；
- 支持 Fabric 多版本构建；
- 支持聊天、物品提示、悬浮提示、书本、告示牌、HUD、成就等游戏文本；
- 支持所有模型/API 能力范围内的语言互转；
- 保留翻译缓存、Token 统计、按住显示原文等实用功能。

### 为什么保留这个仓库

保留此仓库是为了方便查看早期代码、旧版 Release 和历史记录。

此仓库不再作为最新版本发布入口。

## English

This repository contains the old **Simple Translation 1.0** codebase and release history.

The actively maintained **SimpleTranslate 2.0** project has moved to:

<https://github.com/baokaixin/SimpleTranslate>

Please use the new repository for downloads, documentation, issues, and future updates.

### What changed in 2.0

- Rebuilt around Minecraft **Component JSON** translation;
- Fabric multi-version builds;
- Translation support for chat, item tooltips, hover tooltips, books, signs, HUD text, advancements, and more;
- Any-language translation supported by the configured model/API;
- Translation cache, token statistics, and hold-to-show-original features.

This repository is kept only as a historical archive for the old 1.0 version.
